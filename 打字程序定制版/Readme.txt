打开exe程序即可运行,按键盘左上角esc即可暂停，按任意键继续

如果想要修改难度，打开Info.txt修改里面的数字即可，
默认如下：
	25	（列数）
	20	（行数）
	1000	（每隔多少毫秒下落一行（必须是整数 1000毫秒 = 1秒））
	50	（分数每达到多少分加速一次）
	0.95	（加速的速率（小数，整数））
	
	不要在Info里面加入任何别的东西，如汉字，或者多加一行数字，有需求可以跟我说，我帮你改

关于加速：如果你按照默认设置：
				当你分数达到50分时，每隔1000*0.95下落一次
					    100分       1000*0.95*0.95
					    150分       1000*0.95*0.95*0.95
				            ...         ...
				（如果你达到了51分，每隔1000*0.95下落一次
				  但是如果你又低于50分了，下落间隔会恢复到上一档次）
				PS：如果加速速率被你改成大于1的话，那么他会下落的越来越慢
